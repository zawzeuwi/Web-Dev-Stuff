Features

- **Basic Calculations**: Addition, subtraction, multiplication, and division
- **Special Functions**:
 
  - Percentage (%) calculations
  - Square root (√) function

- **User Interface**:

  - Clear button (C) to reset the calculator
  - Backspace button to delete the last digit
  - Error handling for invalid operations like division by zero

- **Customization**:

  - Dark mode toggle for comfortable use in low-light environments
  - 5 different background color themes (Default, Blue, Green, Purple, Orange)

- **Keyboard Support**:

  - Numbers: 0-9
  - Operations: +, -, *, /
  - Decimal point: .
  - Percentage: %
  - Enter key: Calculate result
  - Escape key: Clear display
  - Backspace key: Delete last digit
  - 'r' key: Square root

## How to Use

1. **Installation**:
   - Download the three files: `index.html`, `styles.css`, and `script.js`
   - Make sure they're all saved in the same folder
   - Open `index.html` in any modern web browser

2. **Basic Usage**:
   - Click number buttons to input values
   - Click operation buttons (+, -, *, /) to perform calculations
   - Press "=" to see the result
   - Press "C" to clear the display

3. **Special Functions**:
   - **Percentage (%)**: Enter a number, press %, then the result appears
   - **Square Root (√)**: Enter a number, press √, then the square root appears
   - **Backspace (⌫)**: Removes the last digit entered

4. **Customization**:
   - Toggle dark mode using the switch in the settings area
   - Change the background color by selecting a theme from the dropdown menu

5. **Keyboard Shortcuts**:
   - Use your keyboard for faster calculations
   - Press Enter instead of clicking "="
   - Press Escape instead of clicking "C"
   - Press Backspace to delete the last digit
   - Press 'r' for square root function
